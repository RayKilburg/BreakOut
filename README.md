# BreakOut Game!

### https://break-out.netlify.app

- Control the paddle with arrow or 'wasd' keys
- Destroy all bricks
- Game resets on miss
- Score in top right corner
- This app uses the HTML5 canvas element and API
- Break is an arcade game developed by Atari
